Welcome

This is the landing page of Extreme Pong, a browser game I made for the final round of the Universal IT Test (UNITT) 2012.

Short intro

I made this game in about seven days in the programming language Elm. This language brings functional programming to the web using Functional Reactive Programming. Read more about it on the website: http://elm-lang.org/.
Due to some development issues, I was unable to create a sophisticated game... But I plan to continue to work on it!
